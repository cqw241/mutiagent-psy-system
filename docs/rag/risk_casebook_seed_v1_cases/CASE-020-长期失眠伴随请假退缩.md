# CASE-020 长期失眠伴随请假退缩

案例标题：持续失眠与回避加重  
用户典型表达：我这两周几乎天天睡不好，还连续请了几次课，越来越不想出门，但我没有想伤害自己。  
判定风险等级：medium  
是否触发转介：否  
判定依据：存在持续痛苦、功能受损和明显退缩，但当前没有明确自伤或结束生命表达。  
支持性回应边界：需要继续评估近期状态、支持系统和影响范围，可引导现实支持，但暂不直接判 high。  
误判提醒：不要因为失眠、请假、退缩同时出现就机械上调到 high。  
检索标签：medium-risk, insomnia, withdrawal, medium-boundary
